# 사이트명
MeetGo

## 👩‍💻 프로젝트 소개
20대 대학생들을 위한 미팅 주선 프로그램
https://meet-go.vercel.app/
## ⏲️ 개발 기간
- 2024.03.26(화) ~ 2024.04.30(화)

## 🧑‍🤝‍🧑 개발자 소개
- 황인정 : 실시간 채팅, 미들웨어 체인을 이용한 라우트 보호, 메인페이지
- 최희라 : 로그인/회원가입, 마이페이지, 네브 바
- 이혜진 : 미팅룸, 대기방
- 신서연 : 카카오맵 API를 이용한 채팅방 사이드바

## 💻 개발환경
- Next.js
- TypeScript
- supabase
- supabase-realtime
- tanstack-qeury
- zustand
- tailwind
- NextUI
